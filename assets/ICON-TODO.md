# App Icon TODO

The following icon files are needed for the vibes application:

## Required Files

- `icon.icns` - macOS application icon (required for distribution)
- `icon.png` - PNG version for development and README
- `icon.ico` - Windows icon (if cross-platform support is needed)

## Specifications

- Minimum size: 1024x1024 pixels
- Format: PNG with transparency
- Design should align with the vibes brand:
  - Primary color: #f97316 (orange)
  - Secondary color: #ec4899 (pink)
  - Gradient effect encouraged
  - Modern, minimal aesthetic

## Generation

Use tools like:
- [Icon Set Creator](https://apps.apple.com/us/app/icon-set-creator/id939343785)
- [makeappicon.com](https://makeappicon.com/)
- Or manually create with design software
